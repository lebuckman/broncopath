CS 4800 · Spring 2026 · Project Pitch

Smart Campus
Navigator

A mobile app that replaces CPP's outdated interactive map — giving students real-time building occupancy predictions, classroom availability, and smart route suggestions across campus.

The Problem

CPP's current campus map is static and unintelligent. It can't tell you whether the library is packed right now, which buildings are quietest between classes, or the fastest uncrowded path to your next lecture. Students waste time navigating blindly, showing up to overcrowded study spaces, or wandering unfamiliar buildings. This app fixes that.

Core Features · MVP
🗺️

Campus Map with Live Density

An interactive campus map where every building is color-coded — green, yellow, or red — based on predicted busyness right now.

🔮

Crowd Predictor Engine

Pulls CPP's public class schedule data to calculate how many students are in each building at any given time — no sensors needed.

🚪

Classroom Availability

See which rooms in any building are free right now. Perfect for finding a quiet study spot between classes.

🧭

Smart Route Suggestions

Pick a destination and get a route that avoids the most crowded paths — not just the shortest path.
